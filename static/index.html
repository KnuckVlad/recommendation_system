Музична рекомендаційна система

Цей проект був створений для презентації музичної рекомендаційної системи із застосуванням методів коллабораційної фільтрації та Сингулярного розкладу матриці Сайт є примитивною формою реалізації рекомендаційної системи, що застовує машинне навчання для формування плейлісту користувача на основі вподобань. Список пісень сформован на основі музичних датасетів у вільному доступі. Ласкаво просимо до демо-зони.
